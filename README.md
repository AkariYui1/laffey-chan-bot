# Laffey-chan's bot

Như tên, bot của [Laffey-chan](https://github.com/AkariYui1).

Được sử dụng cho server Discord tạo ra cho người dùng Linux.

## Setup

* Python >=3.11
* Pip

Cài đặt các gói phụ thuộc:

```bash
$ pip install -r pip-requirements.txt
```

Chạy:

```bash
$ python python/Main/main.py
```
